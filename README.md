 PART A: ANN for Regression (California Housing Dataset)

This experiment demonstrates the implementation of an Artificial Neural Network (ANN) for a regression task using the California Housing dataset from Scikit-learn. The dataset is divided into training and testing sets, and the input features are standardized to improve model performance. A feedforward ANN is built using TensorFlow/Keras with multiple hidden layers and trained using the Adam optimizer and Mean Squared Error (MSE) loss function. The trained model predicts median house values, and its performance is evaluated using Mean Squared Error (MSE), Mean Absolute Error (MAE), and Root Mean Squared Error (RMSE), which measure the accuracy of the regression model.

 PART B: ANN for Classification (Breast Cancer Wisconsin Dataset)

This experiment demonstrates the implementation of an Artificial Neural Network (ANN) for a binary classification task using the Breast Cancer Wisconsin dataset from Scikit-learn. The dataset is preprocessed by splitting it into training and testing sets and standardizing the input features. An ANN model is designed using TensorFlow/Keras with hidden layers and a sigmoid activation function in the output layer to classify tumors as malignant or benign. The model is trained using the Adam optimizer and Binary Cross-Entropy loss function. Its performance is evaluated using Accuracy, Precision, Recall, F1-score, and the Confusion Matrix to measure the effectiveness of the classification model
